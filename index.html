Sequence Memory Challenge!

How To Play

The computer will show you a sequence of random digits.

Memorize the numbers and type the entire sequence when prompted.

Each round adds more digits to the sequence, so each round gets harder!

If you make a mistake or quit, booooo.

If you get it correct, yay!

Pass five rounds to win!
