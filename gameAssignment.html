Sequence Memory Challenge!

How To Play

The computer will show you a sequence of random digits. Memorize the numbers and type the entire sequence when prompted. Each round adds more digits to the sequence, so each round gets harder! If you get it correct, yay! If you make a mistake or quit, booooo. Pass five rounds to win!
